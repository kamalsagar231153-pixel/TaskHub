# 🚀 TaskHub – Task Management System

TaskHub is a web-based task management platform designed to help organizations manage employee tasks efficiently. It provides a centralized system where administrators can assign tasks and employees can track and submit their work.

---

## 📌 Project Overview

Many organizations struggle with task management using emails, spreadsheets, or messaging apps. TaskHub solves this problem by providing a structured system for assigning, tracking, and submitting tasks. :contentReference[oaicite:0]{index=0}

The platform improves productivity, transparency, and workflow efficiency within teams.

## 🏗 System Architecture

TaskHub follows a **3-tier architecture**:

1️⃣ **Presentation Layer** – React.js frontend  
2️⃣ **Application Layer** – Node.js + Express backend  
3️⃣ **Data Layer** – MongoDB database  

Communication between frontend and backend occurs through REST APIs. :contentReference[oaicite:3]{index=3}

## 📊 Project Workflow

1. Admin logs into the system  
2. Admin creates and assigns tasks  
3. Employees log in and view tasks  
4. Employees submit their work  
5. Admin reviews submissions  
6. Task status gets updated  

This workflow ensures transparency and efficient monitoring. :contentReference[oaicite:5]{index=5}

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/kamalsagar231153-pixel/TaskHub.git
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Run Backend

```bash
npm run server
```

### Run Frontend

```bash
npm run dev
```
